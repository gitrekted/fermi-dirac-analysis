\# Fermi-Dirac Distribution and Tanh Approximation



This project provides a Python implementation for analyzing the Fermi-Dirac distribution and approximating it using a differentiable tanh-based model. It includes:



\- Exact evaluation of the Fermi-Dirac function

\- First and second derivative analysis

\- Curve fitting using a tanh approximation

\- Error analysis and regression validation



\## Requirements



\- numpy  

\- matplotlib  

\- scipy



Install dependencies using:



```



pip install numpy matplotlib scipy



```



\## Usage



Run the script using:



```



python fermi\\\_dirac\\\_analysis.py



```



The script will generate:



\- Plots of the Fermi-Dirac distribution at various temperatures  

\- Derivatives with respect to energy  

\- Fitted tanh curves and error plots  

\- Linear regression to validate thermal broadening behavior
